angularjs-oauth2


Simple example of the angularjs-oauth2 plugin

Sample

This sample demonstrates using the angularjs-oauth2 plugin to authorize against the IdentityServer3 STS and calling a protected resource implemented using ASP.Net Web API.

For more information see this blog post.

Installation

bower install angularjs-oauth2 --save

Configuration

    <oauth2 authorization-url="https://localhost:44309/identity/connect/authorize"
                 sign-out-url="https://localhost:44309/identity/connect/endsession"
        sign-out-append-token="true"
        sign-out-redirect-url="http://localhost:9000"
                    client-id="sampleapiclient"
                 redirect-url="http://localhost:9000"
                response-type="id_token token"
                        scope="openid myWebApi">
    </oauth2>
